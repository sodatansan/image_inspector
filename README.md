# 画像検査ツール（image_inspector）

## 概要
製造現場の画像検査自動機で行っていた画像処理と寸法計測を
PythonとOpenCVで再現したツールです。
前職での外観検査自動機の運用経験をもとに設計しました。

## 開発の背景
前職で画像検査の調整（コントラスト・エッジ感度など）を
感覚で行っていたが、それぞれの処理が何をしているか
理解したいと思い、自分でコードを書いて再現しました。

## 動作の流れ
画像読み込み → 自動計測 → 結果表示 → 必要に応じて調整 → 再計測

## 機能

### 前処理（画像調整）
- コントラスト調整（明暗の差を強調）
- 明るさ補正（画像全体の明暗を調整）
- 2値化（白黒の2色にして判定しやすくする）
- メディアンフィルタ（ノイズ除去）
- 膨張・収縮（エッジの太さを調整）

### 寸法計測
- エッジ検出（白黒の境目を検出）
- エッジ間の距離を計算（ピクセル単位）
- 倍率をかけて実寸法に変換（mm単位）

### その他
- 設定の保存・読み込み（前回の調整を引き継ぎ）
- 調整後に画像がリアルタイム更新

## 使用技術
- Python
- OpenCV

## 想定製品・検査条件
検査対象：金属部品
検査項目：ネジ径（1箇所）
公差設定：上限10.05mm / 下限9.95mm
※本ツールで使っている製品は架空の製品です
